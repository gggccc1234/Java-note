# Java学习笔记

## 新数据类型
+ byte  -2^7~2^7-1
+ short  -2^15~2^15-1
+ int  -2^31~2^31-1
+ long  -2^63~2^63-1  后缀一定要加L
+ float  8位有效数字  后缀一定要加F
+ double  16位有效数字   后缀可选加D
+ 输入输出数据
    + Scanner 类
    + System.out.print
    + System.out.println  会换行
    + %d
    + %c
    + %f
    + %s
    + %md
    + %m.nd
    + 输入格式类似 C 的printf
+ 数组
    + int [] a
    + int a[][]
    + int [] a,b[]  声明了一个一维数组和二维数组
    + 不能在数组内声明数组的长度
